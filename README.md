# cppd - C++代码混淆器
使用`vue+vite`框架  
实例: [https://dffxd-suntra.github.io/cppd](https://dffxd-suntra.github.io/cppd)  
在信友队编写而成,对于普通的C++程序的解析是没有问题的  
程序没有处理`#ifdef`的头文件,但对于竞赛代码足够了  
部署在子目录请修改根目录下`vite.config.js`中的`base`变量  

## 构建:
``` bash
npm install
npm build
```

## 本地预览
``` bash
npm install
npm preview
```
